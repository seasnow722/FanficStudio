# 開発メモ

# 作業開始時
1.VScodeを開く
2.ファイル＞フォルダーを開くでFanficStudioを開く
3.ターミナル＞新しいターミナルでターミナルを開く
4.ターミナルに下記を入力する
    npm run dev
5.ターミナルに表示されるlocalhost:5173を開く
6.作業開始

# 作業終了時
1. git status
2. git add .
3. git status
4. git commit -m "今日やったこと"
