# Fanfic Studio

Fanfic Studio は、二次創作小説を書くための創作支援ツールです。

「資料を探す時間」を減らし、「書く時間」を増やすことを目的として開発しています。

---

## コンセプト

> 「小説を書くIDE」

VSCodeやObsidianのように、執筆・資料管理・設定管理を一つの画面で行える環境を目指しています。

---

## 主な機能

* 小説本文の執筆（CodeMirrorエディタ）
* 辞書ページによる設定管理
* 原作設定と作品設定をレイヤー管理
* 原作出典・作品注釈の管理
* JSON形式での保存・読み込み
* 小説本文のtxt書き出し
