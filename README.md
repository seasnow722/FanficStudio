# Fanfic Studio
## スクリーンショット


URLはこちら
`https://seasnow722.github.io/FanficStudio/`

現状データはブラウザ内保存です。定期的なJSON書き出しを推奨します。


Fanfic Studio は、二次創作・一次創作を問わず、
「設定」「本文」「物語構成」「作者メモ」「執筆記録」を一つの場所で管理する創作支援ツールです。

世界観やキャラクター設定を辞書として管理しながら、小説本文を書き、時系列やTODOを整理し、執筆ログも記録できます。

「書く」と「考える」を行き来しやすいことを目的に設計しています。

二次創作での利用は想定していますが、公式作品本文・有料コンテンツ・公式資料などを、そのまま保存・共有・公開することは推奨していません。

著作権や利用規約を尊重した範囲でご利用ください。

---

## コンセプト

> 「小説を書くIDE」

VSCodeやObsidianのように、執筆・資料管理・設定管理を一つの画面で行える環境を目指しています。
小説を書くことだけでなく、「考える」「管理する」まで含めた創作環境を目標にしています。

Fanfic Studioでは、
『編集を邪魔しない』『必要な情報へすぐ飛べる』
『「考える」と「書く」を切り替えやすい』
ことを重視しています。

例えば、
・時系列は「編集」と「閲覧」を分離
・辞書は編集しながら閲覧できる
・TODOは一覧で素早く確認・編集できる
など、それぞれ用途に合わせたUIを採用しています。

## 起動方法

### Windows 1
1. `https://seasnow722.github.io/FanficStudio/`をブラウザで開く

### Windows 2
1. Node.jsをインストールする
2. このリポジトリをZIPでダウンロードして解凍する
3. `start-fanfic-studio.bat` をダブルクリックする
4. 表示された `http://localhost:5173` をブラウザで開く

## 主な機能

* 小説本文の執筆（CodeMirrorエディタ）
* 辞書ページによる設定管理
* 設定資料設定と作品設定をレイヤー管理
* 設定資料出典・作品注釈の管理
* 時系列管理
* TODO・作者メモ
* 執筆プロフィール
* JSON形式での保存・読み込み
* 小説本文のtxt書き出し
