# Fanfic Studio 開発引き継ぎ（2026-06-29）

## プロジェクト概要

Fanfic Studioは、二次創作小説を書くための創作支援ツール。

コンセプトは

> **「小説を書くIDE」**

Notion・Obsidian・VSCodeのような操作感を目指し、

* 原作設定
* 作品ごとの差分
* 小説本文
* 時系列
* フラグ

を一つのアプリ内で管理できることを目的としている。

---

# 開発方針

## 最重要思想

### 編集モードを作らない

常に編集できるUIを目指す。

必要なボタンだけホバー時などに表示する。

### CodeMirrorを中心に設計する

textareaは順次廃止。

本文・辞書・将来的には時系列やフラグもCodeMirrorへ移行予定。

### 行単位で管理する

辞書本文は単なる文字列ではなく

```
page.body
page.lineIds
```

で管理する。

各行に永続ID(lineId)を持たせることで、

* 原作出典
* 作品注釈
* 将来の差分管理

を安定して行えるようにしている。
